# Basic Network Sniffer

## 📌 Project Overview

The Basic Network Sniffer is a Python-based cybersecurity tool developed using Scapy. It captures live network packets and displays important information such as the source IP address, destination IP address, protocol, packet length, and payload (when available).

This project was developed as part of the CodeAlpha Cyber Security Internship to demonstrate an understanding of network packet analysis and Python programming.

---

## ✨ Features

- Capture live network packets
- Display Source IP Address
- Display Destination IP Address
- Detect TCP and UDP protocols
- Show packet length
- Display packet payload (when available)
- Simple and easy-to-read output

## ⚙ Installation

1. Clone the repository

```bash
git clone https://github.com/Ayomikun2008/CodeAlpha_BasicNetworkSniffer.git
```

2. Navigate into the project folder

```bash
cd CodeAlpha_BasicNetworkSniffer
```

3. Create a virtual environment

```bash
python3 -m venv venv
```

4. Activate the virtual environment

```bash
source venv/bin/activate
```

5. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶ How to Run

```bash
sudo ./venv/bin/python sniffer.py
```
